# Shebang

&nbsp;

- `#!` = sharp(#) + bang(!) 의 합성어
- 2byte의 매직넘버(magic number)
- 해당 스크립트를 실행시켜줄 프로그램의 경로를 지정하는 역할

- 시스템 환경에 따라 프로그램의 경로가 달라지는 경우 `env`를 사용할 수 있다
  - "`#!/usr/bin/env` + `php`" 식으로 입력
