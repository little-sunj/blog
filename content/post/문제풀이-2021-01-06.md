# 2021.01.05 연습

- 사용언어 : java

```java

class Solution {
    public int maximumWealth(int[][] accounts) {
        
        int result = 0;
        
        for (int[] i :  accounts) {
            int check= 0;
            for (int j : i) {
                check += j;
            }
            result = result < check ? check : result;
        }
        return result;
    }
}

```
