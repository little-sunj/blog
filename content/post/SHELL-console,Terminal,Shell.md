# Console / Terminal / Shell

&nbsp;

- **콘솔**
    - 서버의 로컬장치에서 직접 명령어를 작성. **물리적 터미널**
- **터미널**
    - 서버의 **로컬 또는 원격**으로 접속할 수 있는 콘솔을 구현한 소프트웨어. 쉘을 실행하기 위한 wrapper프로그램. (TTY : teletypewriter)
- **쉘**
    - 실제로 명령어를 전달하고 결과를 전달받는 프로그램. 명령어를 OS로 전달하는 명령어 해석기
    - `$` : normal user
    - `#` : root
    - `%` : C쉘일때 프롬프트 표시
