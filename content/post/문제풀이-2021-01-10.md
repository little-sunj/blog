# 2021.01.10 연습

&nbsp;

- 사용언어 : java


```java

class Solution {
    public int singleNumber(int[] nums) {

        Map<Integer, Integer> map = new HashMap<>();
        for (int i : nums) {
            map.put(i, map.getOrDefault(i, 0) + 1);
        }
        
        for(int i : nums) {
            if(map.get(i).equals(1)) { 
                return i;
            } 
        }

        return 0;
    }
}

```
