# 2021.03.01 연습

&nbsp;

- 사용언어 : java   
- 프로그래머스 레벨2 전화번호 목록

&nbsp;


```java

import java.util.*;

class Solution {
    public boolean solution(String[] phone_book) {
        boolean answer = true;
        //정렬 후 바로 다음것과 비교한다.
        Arrays.sort(phone_book);
        for (int i = 0; i < phone_book.length-1; i++) {
            if (phone_book[i+1].startsWith(phone_book[i])) {
                return false;
            }
        }

        return answer;
    }
}


```
