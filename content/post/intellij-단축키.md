# Intellij 단축키

&nbsp;


### 코드작성

&nbsp;

- 메인메서드 선언
 
```
psvm
```

&nbsp;

- 줄 복사
```
ctrl + d
```

&nbsp;

- 줄 삭제
```
ctrl + y
```

&nbsp;

- 줄 이동
```
alt + shift + 위아래 화살표키
ctrl + shift + 위아래 화살표키 (해당 메서드 안에서만 이동)
```

&nbsp;

- 현재 파일내에서 replace
```
ctrl + r
```

&nbsp;

- 전체에서 replace 
```
ctrl + shift + r
```

&nbsp;

- 자동완성
```
ctrl + space
ctrl + shift + space (스마트 자동완성)
```

&nbsp;

- extract method
```
ctrl + alt + m
```

&nbsp;

- iteration for문 자동완성
```
iter + tab
```

&nbsp;

- `System.out.println` 자동완성
```
sout
soutv
```

&nbsp;

-----

### 검색

&nbsp;

- 전체에서 검색 (find in path)
```
ctrl + shift + f
```

&nbsp;

- 파일검색
```
ctrl + shift + n
```

&nbsp;

- 메서드검색
```
ctrl + shift + alt + n
```

&nbsp;

- 퀵서치
```
shift + shift
```
